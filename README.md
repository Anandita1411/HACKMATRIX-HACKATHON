# Period Tracker App (Lunar Flo)
  Overview

Lunar Flo is a mobile application designed to help users track their menstrual cycle, predict upcoming periods, and monitor symptoms. The app provides a simple and user-friendly interface for better health awareness and planning.

## Features

*  Track menstrual cycle dates
*  redict next period using cycle length
*  Log daily symptoms (mood, cramps, energy)
*  Get reminders for upcoming periods
*  View cycle insights and history

##  Dataset

This application uses **user-generated data** instead of a static dataset.
Data stored includes:

* Last period date
* Cycle length
* Symptoms and mood

 System Workflow

1. User signs up/logs in
2. Enters last period date and cycle length
3. App calculates next period date
4. Data is stored in database
5. Notifications are sent before the next cycle
6. Dashboard displays predictions and insights
